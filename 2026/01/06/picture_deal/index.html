哈希表

1. 两数之和（Two Sum）

思路：

  • 用哈希表 value -> index
  • 遍历数组，查找 target - x 是否已出现

复杂度：

  • 时间：O(n)
  • 空间：O(n)

代码：

class TwoSum:
    def two_sum(self, nums, target):
        num_map = {}
        for i, x in enumerate(nums):
            need = target - x
            if need in num_map:
                return [num_map[need], i]
            num_map[x] = i
        return []
