1. ALM算法

ALM是一种聪明的“解决条件下最优答案”的数学方法，当我们不能随便优化（比如受限制），它能帮我们找到既满足条件又最好的方案，并用于恢复模糊照片、重建医学图像、分离视频背景、压缩感知等场景，以求达到自动、稳定、数学严谨，计算机容易做的目的。

本文将从拉格朗日函数法(Lagrangian Multipliers)讲起，逐步过渡到增广拉格朗日函数法(ALM)，最后讲解ALM优化算法，包括：ADMM算法、AMA算法，并进行四种算法的总结比较。

1.1 拉格朗日函数法

在讲解LM之前，可以回顾高中是否学习过带约束的最优化问题，高中常见的做法是将约束条件进行变量代换，然后带入需求解边界的函数，接着对函数求导，寻找边界极值，进而确定此方程组当中最优化解。

由此可见，高中的求解方式依赖 变量代换，但是在较为复杂的问题当中（如：多变量，多约束），可能无法进行有效的变量代换，此时需要更加普适的求解方式，那就是 LM。

在本小节当中，首先将讲解LM算法的数学求解过程，然后通过一个简单的带等式约束的优化问题，进行印证，并附上python实现代码，紧接着通过一个实际工程问题图像去噪进行理论实践，最后总结LM算法的优缺点，进而完成本次LM算法的学习。

🆕 1.1.1 理论数学推导

设目标函数为：

[
\min_{x\in \mathbb{R}^{n} }f(x)\quad subject; to\quad g(x)=0
]

其中：

  • $f(x)$是最小化的目标函数
  • $g(x)$是等式约束（可以是一个或多个）

此时引入一个拉格朗日乘子$\lambda$，构建拉格朗日函数：
[
\mathcal{L}(x, \lambda) = f(x) + \lambda \cdot g(x)
]
在最优点，函数和约束条件的梯度要”方向一致”，即：
$$
\begin{gathered}
\nabla_{x}\mathcal{L}(x, \lambda) = \nabla f(x) + \lambda \cdot \nabla g(x) = 0 \
g(x) = 0
\end{gathered}
$$
此时就构成了一个联立方程组，用来解出$x$和$\lambda$。

❓ 1.1.2 数学问题的提出

求函数 $f(x, y) = x^2 + y^2$ 的最小值，要求满足约束条件 $x + y = 1$。

✅ 求解步骤
① 构造拉格朗日函数：

[
\mathcal{L}(x, y, \lambda) = x^2 + y^2 + \lambda (x + y - 1)
]

② 对每个变量求偏导并令其为 0：

$$
\begin{aligned}
\frac{\partial \mathcal{L}}{\partial x} &= 2x + \lambda = 0 \
\frac{\partial \mathcal{L}}{\partial y} &= 2y + \lambda = 0 \
\frac{\partial \mathcal{L}}{\partial \lambda} &= x + y - 1 = 0
\end{aligned}
$$

③ 解这个方程组：
  • 由前两式得：$x = y$
  • 代入第三式得：$x + x = 1 \Rightarrow x = y = 0.5$
  • 再代回：$\lambda = -2x = -1$
✅ 最终结果

  • 极小值点为：
    [
    (x, y) = (0.5, 0.5)
    ]

  • 极小值为：
    [
    f(0.5, 0.5) = 0.5^2 + 0.5^2 = 0.5
    ]
