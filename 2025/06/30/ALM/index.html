ALM优化

发表于2025-06-30|更新于2025-07-03|图像去噪
|浏览量:

ALM是一种聪明的“解决条件下最优答案”的数学方法，当我们不能随便优化（比如受限制），它能帮我们找到既满足条件又最好的方案，并用于恢复模糊照片、重建医学图像、分离视频背景、压缩感知等场景，以求达到自动、稳定、数学严谨，计算机容易做的目的。稀疏性（L1）和低秩性是两大结构性先验，其分别刻画了“少而精”的变量（如 LASSO）与“冗余低维”的矩阵（如图像补全），故而本文结构如下：

本文将从拉格朗日函数法(Lagrangian Multipliers)讲起，逐步过渡到增广拉格朗日函数法(ALM)，最后讲解ALM优化算法，并聚焦于L1 优化与低秩优化中的应用。

1.1 拉格朗日函数法

在讲解LM之前，可以回顾高中是否学习过带约束的最优化问题，高中常见的做法是将约束条件进行变量代换，然后带入需求解边界的函数，接着对函数求导，寻找边界极值，进而确定此方程组当中最优化解。

由此可见，高中的求解方式依赖 变量代换，但是在较为复杂的问题当中（如：多变量，多约束），可能无法进行有效的变量代换，此时需要更加普适的求解方式，那就是 LM。

在本小节当中，首先将讲解LM算法的数学求解过程，然后通过一个简单的带等式约束的优化问题，进行印证，并附上python实现代码，紧接着通过一个实际工程问题图像去噪进行理论实践，最后总结LM算法的优缺点，进而完成本次LM算法的学习。

🆕 1.1 理论数学推导

设目标函数为：

min⁡x∈Rnf(x)subject tog(x)=0\begin{gathered} \min_{x\in \mathbb{R}^{n} }f(x)\quad subject\text{ }to\quad g(x)=0 \end{gathered} x∈Rnmin​f(x)subject tog(x)=0​

其中：

  • f(x)f(x)f(x)是最小化的目标函数
  • g(x)g(x)g(x)是等式约束（可以是一个或多个）

此时引入一个拉格朗日乘子λ\lambdaλ，构建拉格朗日函数：

L(x,λ)=f(x)+λ⋅g(x)\mathcal{L}(x, \lambda) = f(x) + \lambda \cdot g(x) L(x,λ)=f(x)+λ⋅g(x)

在最优点，函数和约束条件的梯度要"方向一致"，即：

∇xL(x,λ)=∇f(x)+λ⋅∇g(x)=0\nabla_{x}\mathcal{L}(x, \lambda) = \nabla f(x) + \lambda \cdot \nabla g(x) = 0 ∇x​L(x,λ)=∇f(x)+λ⋅∇g(x)=0

g(x)=0g(x) = 0 g(x)=0

此时就构成了一个联立方程组，用来解出xxx和λ\lambdaλ。

❓ 1.2 数学问题的提出

求函数 f(x,y)=x2+y2f(x, y) = x^2 + y^2f(x,y)=x2+y2 的最小值，要求满足约束条件 x+y=1x + y = 1x+y=1。

✅ 求解步骤

① 构造拉格朗日函数：

L(x,y,λ)=x2+y2+λ(x+y−1)\begin{gathered} \mathcal{L}(x, y, \lambda) = x^2 + y^2 + \lambda (x + y - 1) \end{gathered} L(x,y,λ)=x2+y2+λ(x+y−1)​

② 对每个变量求偏导并令其为 0：

∂L∂x=2x+λ=0\frac{\partial \mathcal{L}}{\partial x} = 2x + \lambda = 0 ∂x∂L​=2x+λ=0

∂L∂y=2y+λ=0\frac{\partial \mathcal{L}}{\partial y} = 2y + \lambda = 0 ∂y∂L​=2y+λ=0

∂L∂λ=x+y−1=0\frac{\partial \mathcal{L}}{\partial \lambda} = x + y - 1 = 0 ∂λ∂L​=x+y−1=0

③ 解这个方程组：

  • 由前两式得：x=yx = yx=y
  • 代入第三式得：x+x=1⇒x=y=0.5x + x = 1 \Rightarrow x = y = 0.5x+x=1⇒x=y=0.5
  • 再代回：λ=−2x=−1\lambda = -2x = -1λ=−2x=−1

✅ 最终结果

  • 极小值点为：

(x,y)=(0.5,0.5)\begin{gathered} (x, y) = (0.5, 0.5) \end{gathered} (x,y)=(0.5,0.5)​

  • 极小值为：

f(0.5,0.5)=0.52+0.52=0.5\begin{gathered} f(0.5, 0.5) = 0.5^2 + 0.5^2 = 0.5 \end{gathered} f(0.5,0.5)=0.52+0.52=0.5​

1.4 图像去噪数学模型建立

假设由于相机感光器出现噪点导致拍摄的带噪声的图像为yyy，需要从噪声图像当中尽可能还原出原始的清晰图像xxx，此时构建图像去噪数学模型如下：

min⁡x∣∣x−y∣∣22subject to∣∣Dx∣∣1≤τ\min_{x} ||x-y||_{2}^{2} \quad subject \text{ } to \quad ||Dx||_{1} \le \tau xmin​∣∣x−y∣∣22​subject to∣∣Dx∣∣1​≤τ

使用ML算法，图像去噪优化问题变为：

min⁡x∣∣x−y∣∣22+λ∣∣Dx∣∣1\begin{gathered} \min_{x}||x-y||_{2}^{2} + \lambda ||Dx||_{1} \end{gathered} xmin​∣∣x−y∣∣22​+λ∣∣Dx∣∣1​​

目标函数是保真项，约束条件是“正则化约束”保证图像的变化程度
具体为何采用此方式完成此任务建模，原因如下：(将依次讲解保真项，正则项以及调节稀疏程度的参数 $ \tau $)

🧠 将总结为以下几个问题进行解释：
❓问题1：图像去噪是什么东西，为什么说能够"还原"图像？
噪声是不可控的，但图像本身的结构是有规律的，对于噪声而言，其往往是高频、随机跳动的，但是自然图像一般而言是低频平滑 + 少量边缘。
❓问题2：怎么用数学建模“图像去噪”？
step1：还原图像要和原图“接近”
此时优化目标为第一项，保真项，即图像重建误差最小，此时使用L2lossL2 lossL2loss(二范数)，进行表示重建前后每一个对应像素之间的误差最小：

min⁡x∣∣x−y∣∣22=min⁡x∑i(xi−yi)2\begin{gathered} \min_{x}||x-y||_{2}^{2} = \min_{x}\sum_{i}(x_{i}-y_{i})^{2} \end{gathered} xmin​∣∣x−y∣∣22​=xmin​i∑​(xi​−yi​)2​

step2：不希望结果里有“噪声”
即不希望图片的变化过于剧烈，此时图像应当是“平滑”的，此时引入衡量图像变化程度的指标图像梯度（像素间的差值），其计算公式如下：

∣∣Dx∣∣1||Dx||_{1} ∣∣Dx∣∣1​

其中DDD表示差分算子（用来计算图像像素之间的变化），∣∣Dx∣∣1||Dx||_{1}∣∣Dx∣∣1​表示所有像素之间的差值绝对值之和（图像变化总量），值很小 ⇒\Rightarrow⇒ 图像是平滑的，值很大 ⇒\Rightarrow⇒ 图像中变化很多，可能是噪声引起。

🆕 解释二维差分算子(适合图像的) DDD
作用：衡量“相邻像素之间差异”的工具
本质：“求导的离散版本”，需要保证稀疏性（即：0很多）
操作：分别计算水平方向差分DxD_{x}Dx​(计算每个像素与右边像素的差)以及垂直方向差分DyD_{y}Dy​(计算每个像素与下方像素的差),注意：做差的方向只要规定好且统一即可，只用能够体现出变化差异，但是一般使用前向差分即∇x≈xi+1−xi\nabla x \approx x_{i+1}-x_{i}∇x≈xi+1​−xi​
例子：针对图像为m×nm \times nm×n(高度 x 宽度)，展平成向量为x∈Rmnx \in \mathbb{R}^{mn}x∈Rmn,此时的差分操作使用矩阵表示为：

∇x=[DxDy]×x⇒{xi,j+1−xi,jxi+1,j−xi,j\begin{gathered} \nabla x = \begin{bmatrix} D_{x} \\ D_{y} \end{bmatrix} \times x \Rightarrow \left\{\begin{matrix} x_{i,j+1} - x_{i,j}\\ x_{i+1,j} - x_{i,j} \end{matrix}\right. \end{gathered} ∇x=[Dx​Dy​​]×x⇒{xi,j+1​−xi,j​xi+1,j​−xi,j​​​

其中Dx∈Rm(n−1)×mnD_{x} \in \mathbb{R}^{m(n-1) \times mn}Dx​∈Rm(n−1)×mn , Dy∈Rn(m−1)×mnD_{y} \in \mathbb{R}^{n(m-1) \times mn}Dy​∈Rn(m−1)×mn

具体而言：

❓问题3：为什么约束条件使用的正则项是L1L1L1范数
由于第二范数在最小化误差像素，所以其追求的是“整体平滑”，但是使用第二范数，其具有稀疏性，此时其好保留了关键边缘，消除了区域内部的噪声(区别在于：L2L2L2正则会试图减小所有元素（使它们都变小），但不会让很多为 0，但是L1L1L1正则会直接把多数元素“压扁”成 0 → 稀疏！，这个过程发生在优化当中optimizer)，也就是说L2L2L2保证还原观测图像，L1L1L1 让图像变稀疏（平滑 + 保留边缘），二者搭配，在优化器作用下实现高质量去噪。

❓问题4：如何取得一个合适的调节稀疏程度的参数τ\tauτ？
一般来说，我们采用交叉验证的方法进行寻找合适的τ\tauτ的值，即：遍历多个τ\tauτ值（如 10、50、100、200、500），对每个值执行去噪，然后评估结果，基于：PSNR、SSIM 或视觉观察。
